Portfolio Website Project

This portfolio website project is a collaborative effort that leverages UI/UX design principles to create an engaging and professional online presence. Developed by Benjamin Ghajiga and Bernice Lawson, it showcases a comprehensive range of skills including web development, ethical hacking, videography, photography, and graphic design.

The website features a modern, responsive design with smooth navigation, interactive elements, and optimized performance across all devices. Key sections include an introduction, project showcase, about page, and contact form with integrated social media links and messaging options.

Technologies and tools used: HTML5, CSS3, JavaScript, responsive design frameworks, and version control systems. The project demonstrates expertise in front-end development, user experience design, and collaborative coding practices.
